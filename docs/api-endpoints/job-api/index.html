Job Handling

This category is related to creating, retrieving, and managing rendering jobs.

Create Job​

Endpoint: POST /api/v1/job

Create a new rendering job.

Request​

  • Method: POST
  • Body: JSON raw data
{
    "webhook": {
        "url": "URL_WHERE_TO_SEND_AFTER_RENDERING",
        "secret": "BEARER_TOKEN_IF_REQUIRED"
    },
    "creation_data": {
        "image_1": "64c4466cef34a47228f27c9d",
        "image_2": "64c4466cef34a47228f27c9d",
        "audio_name": "64c46b64ef34a47228f27cb0",
        "audio_age": "64c46b74ef34a47228f27cb2",
        "audio_hobby": "64c46b88ef34a47228f27cb4",
        "audio_behavior": "64c46b9aef34a47228f27cb6",
        "audio_advice": "64c46bbbef34a47228f27cb8"
    },
    "scenario_id": "12345"
}

Response​

  • Status Code: 200 OK (on success)
